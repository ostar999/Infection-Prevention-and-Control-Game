# 🏥 传染病防护知识培训游戏

基于《常见传染病和感染性疾病的防护要求》开发的交互式培训工具，面向医护人员及感控相关人员，通过游戏化方式帮助掌握 24 种常见传染病的防护装备选择、传播途径分类和隔离方式判断。

## 使用

用浏览器打开 `index.html` 即可，无需安装、无需网络。数据存储在浏览器 localStorage。

## 功能

### 三种训练模式

| 模式 | 说明 | 题数 | 计时 |
|------|------|------|------|
| 🛡️ 防护装备选择 | 根据疾病选正确的基本+可选防护装备 | 5 | 不限时 |
| 🦠 传播途径分类 | 将疾病归类到 6 种隔离类型 | 10 | 可开关（默认关） |
| ⚡ 快问快答 | 单选/判断/多选题混合 | 10 | 可开关（默认关） |

### 学习工具

- **📊 学习统计** — 累计得分/答题/正确率/连击 + 疾病维度掌握度表格
- **📝 错题本**（独立导航） — 错题记录 + 错题重练（按错题疾病针对性出题）
- **📖 知识速查**（固定表头+内部滚动）— 24 种疾病完整对照表 + 搜索

### 界面

- **左右分栏布局** — 左侧边栏（品牌/导航/统计/主题控制），右侧答题区
- **4 套主题** — 默认蓝 / 暖金 / 森绿 / 暗紫，左上角下拉切换
- **暗色模式** — 独立开关，与主题叠加
- **响应式** — ≤768px 自动切换上下布局

### 游戏机制

- **连击系统**：连续答对累计连击，模式三连击越高加分越多
- **评分**：正确加分、错误/遗漏扣分
- **数据管理**：JSON 导出/导入，支持备份迁移
- **评级**：🏆优秀 / 👍良好 / 📚加油 / 💪需努力

## 技术

- 单文件 HTML/CSS/JS，无框架依赖
- CSS 变量驱动主题系统，`body.dark` 暗色叠加
- localStorage 持久化

### .gitignore 是什么？

`.gitignore` 告诉 Git **哪些文件不需要版本管理**。不是所有文件都需要上传到 GitHub，有些文件：
- 体积很大且可以重新生成（如 `.venv/`、`node_modules/`）
- 包含个人本地配置（如 `.vscode/`）
- 是操作系统自动生成的（如 `.DS_Store`）
- 是临时或备份文件（如 `backups/`）

把它们写进 `.gitignore`，Git 就会自动忽略它们，保持仓库干净。

| 忽略规则 | 类型 | 为什么忽略 |
|----------|------|-----------|
| `.venv/`、`venv/` | Python 虚拟环境 | 体积大，每人自己 `python -m venv .venv` 创建 |
| `node_modules/` | Node 依赖 | 可随时 `npm install` 重新下载 |
| `__pycache__/`、`*.pyc` | Python 缓存 | 自动生成，不需要版本管理 |
| `backups/` | 备份文件 | 原始数据副本，非项目运行必需 |
| `.DS_Store`、`Thumbs.db` | 系统文件 | macOS/Windows 自动生成的无关文件 |
| `.vscode/`、`.idea/` | 编辑器配置 | 个人工具配置，不适合共享 |
| `.claude/` | AI 工具配置 | Claude Code 本地数据，不适合共享 |
| `*.log`、`*.tmp` | 临时文件 | 日志和临时数据 |
| `*.zip`、`*.tar.gz` | 压缩包 | 不应提交二进制归档文件 |
